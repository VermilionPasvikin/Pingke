## 文档目的

本文档旨在帮助用户了解并正确使用评课系统，详细介绍系统的各项功能和操作方法。

## 系统概述

评课系统是一个基于微信小程序的课程评价平台，旨在为学生提供便捷的课程和教师评价渠道，同时帮助其他学生选课。系统支持课程查询、评价提交、讨论区交流和排行榜查看等功能。

## 使用前提

1. 手机已安装微信应用
2. 微信小程序版本兼容（推荐使用最新版本）
3. 手机网络连接正常

## 功能介绍

### 1. 进入系统

**操作步骤：**
1. 打开微信应用
2. 扫描系统提供的二维码，或在微信中搜索"评课系统"小程序
3. 进入系统首页

### 2. 课程列表

**功能说明：** 展示系统中的所有课程，支持按学院和学年筛选。

**操作步骤：**
1. 在首页点击"课程列表"或直接在首页浏览
2. 点击顶部筛选按钮，选择学院和学年进行筛选
3. 点击"重置"按钮可清除所有筛选条件
4. 下拉刷新加载最新数据
5. 上拉加载更多课程

### 3. 课程详情

**功能说明：** 展示课程的详细信息，包括课程基本信息、教师信息和评价概况。

**操作步骤：**
1. 在课程列表页点击任一课程卡片
2. 进入课程详情页，浏览课程和教师信息
3. 查看课程评价汇总数据
4. 点击评价入口进入评价详情
5. 点击讨论区入口进入讨论区

### 4. 评价功能

**功能说明：** 查看和提交课程评价。

#### 4.1 查看评价

**操作步骤：**
1. 在课程详情页点击"查看评价"或评价数量区域
2. 进入评价列表页，浏览所有评价
3. 评价默认按时间排序，可切换排序方式
4. 下拉刷新加载最新评价
5. 上拉加载更多评价

#### 4.2 提交评价

**操作步骤：**
1. 在课程详情页点击"我要评价"按钮
2. 进入评价页面，进行以下操作：
   - 拖动星级滑块对课程进行总体评分（1-5分）
   - 对工作量、内容、教学三个维度分别评分
   - 在文本框中输入评价内容（可选）
   - 选择合适的评价标签（可选）
3. 点击"提交评价"按钮完成评价
4. 系统提示评价成功

**注意事项：**
- 每个课程每个用户只能评价一次
- 评价内容应遵守相关规定，不得包含违法违规内容
- 评价提交后不可修改

### 5. 讨论区

**功能说明：** 针对特定课程进行讨论交流。

#### 5.1 查看讨论

**操作步骤：**
1. 在课程详情页点击"讨论区"或讨论数量区域
2. 进入讨论区页面，浏览所有讨论内容
3. 讨论默认按时间排序，可切换排序方式
4. 点击讨论内容可展开查看回复
5. 下拉刷新加载最新讨论
6. 上拉加载更多讨论

#### 5.2 发表讨论

**操作步骤：**
1. 在讨论区页面点击底部"发表讨论"按钮
2. 在输入框中输入讨论内容
3. 点击"发送"按钮提交讨论
4. 系统提示发表成功

#### 5.3 回复讨论

**操作步骤：**
1. 在讨论区页面找到需要回复的讨论内容
2. 点击讨论内容下方的"回复"按钮
3. 在输入框中输入回复内容
4. 点击"发送"按钮提交回复
5. 系统提示回复成功

### 6. 排行榜

**功能说明：** 展示课程的排名情况。

**操作步骤：**
1. 在首页点击"排行榜"入口
2. 进入排行榜页面，默认显示综合排名
3. 点击顶部切换按钮，可查看不同维度的课程排名
4. 排行榜默认显示前20名课程

### 7. 搜索功能

**功能说明：** 根据关键词搜索课程。

**操作步骤：**
1. 在首页或课程列表页点击顶部搜索框
2. 输入搜索关键词（课程名称、教师姓名等）
3. 点击键盘上的"搜索"按钮或系统搜索图标
4. 查看搜索结果
5. 点击搜索结果可进入课程详情页

## 常见问题

### 1. 为什么无法提交评价？

**可能原因：**
- 您已经评价过该课程
- 网络连接异常
- 评价内容不符合要求

**解决方案：**
- 检查是否已经评价过该课程
- 检查网络连接，重新提交
- 确保评价内容符合系统要求

### 2. 如何修改已提交的评价？

目前系统不支持修改已提交的评价，提交前请仔细核对评价内容。

### 3. 为什么搜索不到特定课程？

**可能原因：**
- 课程尚未录入系统
- 搜索关键词有误
- 系统数据更新延迟

**解决方案：**
- 尝试使用不同关键词搜索
- 稍后再尝试搜索

### 4. 讨论区内容发布后是否可以删除？

目前系统暂不支持用户自行删除讨论内容。如内容违规，请联系客服处理。

### 5. 排行榜数据是如何计算的？

排行榜数据基于所有用户的评价综合计算得出，定期更新。
